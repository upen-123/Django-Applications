# Build a service to get to know the detail of the user

# Created 3 API's

1. Registration of User
2. Get User based on mobile number
3. To mark whether user is spam or not

Steps to run

1. Create Virtual Environment
2. Install the requirements (pip install -r requirements.txt)
3. Install the database
4. Run Migrations
4. Environment will be ready and you are ready to execute API
